# Images des Projets

Placez vos images de projets dans ce dossier avec les noms suivants :

- **projet1.jpg** - BuzzBot (Robot)
- **projet2.png** - Serveur WEB (Raspberry Pi)
- **projet3.jpg** - StarFrogger (Jeu)
- **projet4.jpg** - Base de données (Port de La Rochelle)

## Recommandations :

- Format : JPG, PNG ou WebP
- Dimensions recommandées : 400x250 pixels (ou ratio 16:10)
- Taille : < 500 Ko par image pour de meilleures performances

Les images seront automatiquement affichées sur votre portfolio une fois que vous les aurez ajoutées dans ce dossier.
